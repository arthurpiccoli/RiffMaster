# Riff Master
Jogo de música, desenvolvido em trio para a disciplina de Arquitetura e Organização de Computadores I na linguagem Assembly.

Para executar o jogo deve-se utilizar o simulador de MIPS MARS através do arquivo Mars_4_5_Mod.jar.

Como rodá-lo no MARS e demais informações disponíveis no Relatório em formato docx.
